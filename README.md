# nestJS

## NestJS

### start

```
$ npm i -g @nestjs/cli
$ nest new project-name
```

1. npm이 설치된 상태에서 nestJs cli를 설치해 줍니다. -g 옵션은 글로벌 모드입니다. 주로 개발 환경을 설정할때 글로벌 옵션을 많이 설정합니다.

```
$ nest new project-name
```

2. cli가 성공적으로 설치되었다면 cli 명령어로 새 프로젝트를 생성합니다. 저는 nest-js 라는 프로젝트로 명령어를 실행시켰습니다.
